E-Paper Weather Display

Problem

Wanted to create a low-power weather display using e-paper technology with ESP32 microcontroller.

Constraints

V2 Waveshare display had persistent ghosting issues not documented in standard guides.

Solution

Developed custom code in VS Code with PlatformIO, implemented hardware-specific initialization sequences to eliminate ghosting on V2 displays.

Results

Fully functional weather display with crisp, ghost-free updates. Documented V2-specific fixes for the community.

What I Learned

Hardware version differences matter. Importance of thorough documentation and systematic debugging.

Custom CNC Bottle Opener

Project Overview

Custom-designed bottle opener showcasing precision CNC woodworking with decorative inlay techniques.

Fabrication Process

Material Selection: Cherry wood for main body with walnut inlay for visual contrast and durability.

CNC Machining: Precise toolpath planning for clean edges and smooth surfaces with careful attention to feeds and speeds.

  • Selecting appropriate finish for durability and food safety

What I Learned
